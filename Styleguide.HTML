In-text link

In-text links are differenciated from usual paragraphs by size and font-weightd difference. When hovered, it also presents a subtle shade of grey change.

CSS selector:

.in-text-link

Sample code:

<strong><a href="#" class="link">Default link</a></strong>

Rendered element:

In-text-Link

Submit/Reset Buttons

The use of the submit/reset buttons are mostly for customers to proceed through the purchasing process. We decided to use comparatively lower visual weight styling for the buttons so that customers can feel formal and safe when purchasing products. It also has the hover effects of grey out font and border colour.

CSS selector:

.button

Sample code:

<a href="Homepage.html" class="button">Submit</a>

Rendered element:

Submit
